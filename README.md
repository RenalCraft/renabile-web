# 📱 Renabile Web PWA для iPhone (Руководство по установке и деплою)

Этот каталог содержит полноценную, оптимизированную под **iOS Safari** веб-версию (Progressive Web App) чат-приложения **Renabile**, которая может заменять клиент для ПК или Android непосредственно на iPhone и открываться во весь экран без рамок браузера.

---

## Что внутри структуры `/web-pwa`:
1. **`index.html`** — Одностраничный интерфейс, построенный по канонам Material Design 3. Содержит все теги интеграции с iOS standalone-режимом (`apple-mobile-web-app-capable`).
2. **`styles.css`** — CSS-стили, оптимизированные под мобильные экраны iOS (учет физических зон вырезов камер `safe-area-inset-top` и системного индикатора жестов `safe-area-inset-bottom`, плавной физики прокрутки "rubber-banding").
3. **`app.js`** — Вся логика общения с сервером по протоколу WebSocket. Поддерживает регистрацию, адаптивную загрузку истории, оффлайн-оптимизацию, отправку медиафайлов и динамическое управление профилем.
4. **`manifest.json`** — Манифест параметров PWA для добавления на домашний экран.
5. **`sw.js`** — Service Worker, кэширующий файлы приложения на телефоне для мгновенного запуска и работы оффлайн.

## 🚀 1. Как задеплоить Веб-Версию (Хостинг)

Для работы PWA обязателен протокол **HTTPS** (безопасное соединение, без него iPhone не разрешит зарегистрировать Service Worker). Вот три лучших бесплатных пути хостинга вашего веб-клиента:

### Вариант А: Vercel (Рекомендуемый, 2 минуты)
1. Установите Vercel CLI на ПК: `npm install -g vercel` или зарегистрируйтесь на [vercel.com](https://vercel.com).
2. Перейдите в папку веб-PWA:
   ```bash
   cd web-pwa
   ```
3. Выполните команду деплоя:
   ```bash
   vercel
   ```
4. Следуйте инструкциям на экране (согласитесь со всеми настройками по умолчанию). На выходе вы получите бесплатный защищенный адрес вида `https://renabile-chat-pwa.vercel.app`.

### Вариант Б: Netlify (Визуальный деплой без терминала)
1. Зарегистрируйтесь на [netlify.com](https://www.netlify.com).
2. Зайдите в панель управления и перейдите во вкладку **Sites**.
3. Перетащите всю созданную папку `web-pwa` на интерактивную панель загрузки в браузере.
4. Готово! Netlify мгновенно выдаст защищенный SSL-адрес.

### Вариант В: GitHub Pages
Если ваш репозиторий привязан к GitHub:
1. Включите в настройках репозитория вкладку **Pages**.
2. Укажите в качестве источника ветку `main` и директорию `/web-pwa`.
3. Сохраните. Приложение станет доступно по адресу `https://<ваш-логин>.github.io/<имя-репозитория>`.

### Вариант Г: Hosting на Render (как Static Site — БЕСПЛАТНО!)
Так как это веб-PWA (содержит только статические файлы HTML, CSS, JS), его невероятно просто загрузить на **Render.com** совершенно бесплатно:
1. Авторизуйтесь на [Render.com](https://render.com).
2. Нажмите синюю кнопку **New +** в правом верхнем углу и выберите **Static Site** (Статический сайт).
3. Подключите ваш Git-репозиторий с проектом (если проект не на GitHub, его можно загрузить туда).
4. Заполните конфигурацию:
   - **Name**: `renabile-pwa` (или любое другое имя)
   - **Branch**: `main` (или имя вашей рабочей ветки)
   - **Build Command**: Оставьте **пустым** (нам ничего не нужно компилировать)
   - **Publish Directory**: Напишите `web-pwa` *(Это самый важный пункт! Мы указываем Render использовать именно подпапку `web-pwa` как корень сайта, а не весь Android-проект)*
5. Нажмите кнопку **Create Static Site**.
6. Через 30 секунд ваш сайт будет развернут, и Render выдаст бесплатную ссылку с поддержкой HTTPS (например, `https://renabile-pwa.onrender.com`), которую вы сможете открыть на iPhone и добавить на домашний экран.

## ⚙️ 2. Какое серверное окружение необходимо?

Веб-клиент подключается к WebSocket backend-серверу. В проекте уже есть готовый Node.js бэкенд в папке `/backend`.

### Шаги по запуску Backend Сервера (например, на платформе Render/Railway):
1. **База данных PostgreSQL**:
   Сервер хранит аккаунты, списки друзей, статусы, группы и зашифрованную переписку в РСУБД PostgreSQL.
   - Подойдет бесплатная БД на **Render**, **Railway**, **Supabase** или **Neon.tech**.
   - Получите строку подключения вида: `postgres://user:password@hostname:5432/dbname?sslmode=require`.

2. **Хостинг бэкенда (Render.com)**:
   - Зайдите на Render, выберите **New Web Service**.
   - Подключите этот репозиторий.
   - Задайте директорию запуска (Root Directory): `backend` (или оставьте в корне и пропишите команду запуска `npm start`).
   - Настройте **Environment Variables (Переменные окружения)**:
     * `DATABASE_URL` = Ваша строка подключения к базе данных PostgreSQL (обязательно для базы).
     * `PORT` = `8080` (Render подставит порт автоматически).
   - Задеплойте службу. Сервер запустится и будет доступен по безопасному протоколу WebSockets: `wss://<домен-сервера>.onrender.com`.

3. **Связь бэкенда и PWA фронтенда**:
   - При запуске PWA на iPhone в форме входа нажмите кнопку **"Настройки сервера"** ⚙️.
   - Впишите туда ваш адрес WebSocket бэкенда (например, `wss://renabile-node-storage.onrender.com`).
   - Приложение запомнит его в памяти телефона (`localStorage`), и при следующих запусках автоподключение пойдет к вашему серверу.

## 📲 3. Инструкция по созданию иконки приложения

Чтобы PWA смотрелся как настоящее приложение при добавлении на рабочий стол iPhone:
1. Нарисуйте или сгенерируйте квадратную иконку размером **512x512** пикселей в формате `.png`.
2. Сохраните две версии в папку `/web-pwa/icons/`:
   - `/web-pwa/icons/icon-192.png` (сожмите изображение до 192x192)
   - `/web-pwa/icons/icon-512.png` (оставьте исходные 512x512)
3. Создайте фоновое изображение для загрузки (Splash Screen) и поместите как `/web-pwa/icons/splash.png` (необязательно, Safari сгенерирует дефолтный фон на основе `background_color` из манифеста).

## 🍏 4. Как установить приложение на iPhone

1. Откройте браузер **Safari** на вашем iPhone.
2. Перейдите по адресу, на который вы загрузили сайт (например, `https://renabile-chat-pwa.vercel.app`).
3. В нижней панели Safari нажмите кнопку **Поделиться** (квадрат со стрелкой вверх ↑).
4. Прокрутите список вниз и выберите пункт **«На экран "Домой"»** (Add to Home Screen).
5. Подтвердите название приложения (например, "Renabile") и нажмите **«Добавить»** в правом верхнем углу.
6. На вашем рабочем столе появится иконка чата. Запустите её — чат откроется в полноценном автономном окне, изолированном от основного браузера, со встроенным кэшированием и поддержкой оффлайн режима!

## 🎨 5. Фишки и Кастомизация PWA на iOS:
- **Безопасные границы Safe Area**: Верхняя грань адаптируется под "челку" вашего iPhone, окрашиваясь в цвет выбранной темы дизайна.
- **Интегрированная звуковая система**: При активном тумблере "АЛО ПРОСЫПАЙСЯ" в настройках, Safari воспроизводит входящий сигнал звонка, даже если фокус смещен.
- **Кастомные Канал-темы**: Доступен выбор предустановленных тем оформления (Discord Dark, Багровый закат, Cyber Punk). Переключаясь, приложение намертво перекрасит панели телефона.
- **Поддержка push-уведомлений**: Для включения push-уведомлений на iOS перейдите в Настройки системы -> Safari -> Дополнения -> Экспериментальные функции -> включите `Push Notifications` (на iOS 16.4+ они работают из коробки при первом добавлении PWA на экран).
