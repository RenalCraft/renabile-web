# 🚀 Инструкция: Как залить папку `web-pwa` в отдельный GitHub репозиторий через консоль

Если вы хотите выложить **только веб-версию** (содержимое папки `web-pwa`) в отдельный репозиторий на GitHub, чтобы затем развернуть её на **Render** или другом хостинге, следуйте этой простой пошаговой инструкции для терминала (консоли).

---

## Шаг 1: Скачайте проект к себе на компьютер
Вы можете скачать ZIP-архив всего проекта из меню **Google AI Studio** или клонировать текущий репозиторий. Распакуйте архив и найдите папку `web-pwa`.

---

## Шаг 2: Откройте терминал (консоль)
Перейдите в папку `web-pwa` на вашем компьютере.

**В Windows** (используйте Git Bash или командную строку):
```bash
# Перейдите в папку вашего проекта (замените путь на ваш реальный путь)
cd "C:\Users\ИмяПользователя\Downloads\project-name\web-pwa"
```

**В macOS / Linux** (используйте Терминал):
```bash
# Перейдите в папку вашего проекта (замените путь на ваш реальный путь)
cd ~/Downloads/project-name/web-pwa
```

---

## Шаг 3: Инициализация нового Git-репозитория
Прямо внутри папки `web-pwa` создаем независимый локальный Git-репозиторий:

```bash
# 1. Инициализируем Git
git init

# 2. Добавляем все файлы папки в будущий коммит
git add .

# 3. Делаем первый коммит
git commit -m "Initial commit- Web PWA client for iPhone"

# 4. Переименовываем основную ветку в main
git branch -M main
```

---

## Шаг 4: Связь с вашим GitHub
1. Зайдите на свой аккаунт на [github.com](https://github.com).
2. Нажмите зеленую кнопку **New** (Создать новый репозиторий).
3. Дайте ему имя, например: `renabile-web-pwa`.
4. **ВАЖНО:** Оставьте все галочки (Add a README, .gitignore, license) **пустыми**! Нам нужен абсолютно чистый репозиторий.
5. Нажмите **Create repository**.
6. GitHub покажет вам команды для связывания. Скопируйте строчку добавления удаленного репозитория и выполните её в вашей консоли:

```bash
# Добавляем ссылку на ваш репозиторий (замените USER_NAME на ваш ник, а REPO-NAME на имя репозитория)
git remote add origin https://github.com/USER_NAME/REPO-NAME.git
```

*Пример:*
`git remote add origin https://github.com/renalcraft2/renabile-web-pwa.git`

---

## Шаг 5: Пушим файлы (отправка на GitHub!)
Отправьте локальные коммиты на ваш удаленный репозиторий:

```bash
git push -u origin main
```
*Если Git запросит авторизацию — введите ваши данные от GitHub (или используйте Personal Access Token).*

---

## Шаг 6: Деплой на Render.com (Бесплатно!)
Теперь, когда чистый репозиторий с файлами PWA лежит на GitHub, подключить его к Render — дело 30 секунд:

1. Перейдите на [Render.com](https://render.com) и войдите под вашим аккаунтом.
2. Нажмите синюю кнопку **New +** -> выберите **Static Site** (Статический сайт).
3. Connect (Подключите) ваш репозиторий `renabile-web-pwa`.
4. Задайте настройки:
   - **Name**: `renabile-pwa`
   - **Branch**: `main`
   - **Build Command**: Оставьте **пустой** (стираем всё, что там написано)
   - **Publish Directory**: Напишите `.` (просто точку, так как наши файлы лежат прямо в корне репозитория)
5. Нажмите **Create Static Site**.
6. Готово! Render выдаст вам адрес вида `https://renabile-pwa.onrender.com`. Сразу открывайте этот адрес на вашем iPhone в браузере **Safari**, нажимайте кнопку **«Поделиться»** и выбирайте **«На экран "Домой"»**!
